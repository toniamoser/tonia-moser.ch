# Kurzbeschreibung

Mein Projekt ist ein visuell starkes, interaktiv animiertes Portfolio. Der Fokus liegt auf dynamischen Scroll-Animationen (GSAP/ScrollTrigger), handgezeichneten Stickern, Parallax-Effekten und individuell gestalteten Sections, die ein spielerisches, persönliches Erlebnis bieten. Die Seite kombiniert typografische Highlights, bewegte Elemente und modulare Komponenten zu einer flüssigen, lebendigen User-Experience.

# Learnings
Erst Beginnen mit Programmieren, wenn das finale Design steht mit dem man 100% zufrieden ist
Lieber weniger Projekte im Portfolio abbilden, dafür diese genauer beschreiben und die Projekte so wählen, dass eine Vielfalt meiner Kompetenzen klar wird
